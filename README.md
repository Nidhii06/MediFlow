# MediFlow 

MediFlow is a web-based healthcare application designed to help users
check medicine availability and streamline communication between patients
and pharmacies.

## Features
- Medicine search and availability tracking
- Pharmacy-side order and status management
- Simple and intuitive user interface

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express
- MySQL

## Project Status
This project is under active development and is being evolved into a
fully deployable MVP with mobile application support.
